\doxysubsection{Detailed Description}
Event Request Unit (E\+RU) driver for the X\+MC microcontroller family. 

The Event Request Unit (E\+RU) is a versatile multiple input event detection and processing unit. The E\+RU module can be used to expand the P-\/to-\/P connections of the device\+: ports-\/to-\/peripherals, peripherals-\/to-\/peripherals and ports-\/to-\/ports. It also offers configurable logic, that allows the generation of triggers, pattern detection and real-\/time signal monitoring.



The driver is divided into two sections\+: \begin{DoxyParagraph}{Event trigger logic (E\+R\+U\+\_\+\+E\+TL)\+:}
This section of the L\+LD provides the configuration structure X\+M\+C\+\_\+\+E\+R\+U\+\_\+\+E\+T\+L\+\_\+\+C\+O\+N\+F\+I\+G\+\_\+t and the initialization function \mbox{\hyperlink{group___e_r_u_ga0ada9e9c756616d815c573168c841bd4}{X\+M\+C\+\_\+\+E\+R\+U\+\_\+\+E\+T\+L\+\_\+\+Init()}}.~\newline
It can be used to\+:
\begin{DoxyEnumerate}
\item Select one out of two inputs (A and B). For each of these two inputs, a vector of 4 possible signals is available. (\mbox{\hyperlink{group___e_r_u_gad32a6e519fa852eba1895ad9b2cb0448}{X\+M\+C\+\_\+\+E\+R\+U\+\_\+\+E\+T\+L\+\_\+\+Set\+Source()}})
\item Logically combine the two input signals to a common trigger. (\mbox{\hyperlink{group___e_r_u_gad32a6e519fa852eba1895ad9b2cb0448}{X\+M\+C\+\_\+\+E\+R\+U\+\_\+\+E\+T\+L\+\_\+\+Set\+Source()}})
\item Define the transition (edge selection, or by software) that leads to a trigger event and can also store this status. (\mbox{\hyperlink{group___e_r_u_ga95937af5ccdea42b8a96a9e8aa3215dc}{X\+M\+C\+\_\+\+E\+R\+U\+\_\+\+E\+T\+L\+\_\+\+Set\+Edge\+Detection()}} and \mbox{\hyperlink{group___e_r_u_gab3bf06af59ed8e3f28348db8c03b6627}{X\+M\+C\+\_\+\+E\+R\+U\+\_\+\+E\+T\+L\+\_\+\+Set\+Status\+Flag()}})
\item Distribute the events and status flags to the output channels. (\mbox{\hyperlink{group___e_r_u_ga6335e7c97f4834704eb27876741310d7}{X\+M\+C\+\_\+\+E\+R\+U\+\_\+\+E\+T\+L\+\_\+\+Enable\+Output\+Trigger()}})
\end{DoxyEnumerate}
\end{DoxyParagraph}
