\doxysubsection{Detailed Description}
Inter Integrated Circuit(\+I\+I\+C) driver for the X\+MC microcontroller family. 

U\+S\+IC I\+IC Features\+: ~\newline

\begin{DoxyItemize}
\item Two-\/wire interface, with one line for shift clock transfer and synchronization (shift clock S\+CL), the other one for the data transfer (shift data S\+DA) ~\newline

\item Communication in standard mode (100 k\+Bit/s) or in fast mode (up to 400 k\+Bit/s) ~\newline

\item Support of 7-\/bit addressing, as well as 10-\/bit addressing ~\newline

\item Master mode operation, where the I\+IC controls the bus transactions and provides the clock signal. ~\newline

\item Slave mode operation, where an external master controls the bus transactions and provides the clock signal.~\newline

\item Multi-\/master mode operation, where several masters can be connected to the bus and bus arbitration can take place, i.\+e. the I\+IC module can be master or slave. ~\newline
 The master/slave operation of an I\+IC bus participant can change from frame to frame. ~\newline

\item Efficient frame handling (low software effort), also allowing D\+MA transfers ~\newline

\item Powerful interrupt handling due to multitude of indication flags ~\newline
 
\end{DoxyItemize}
