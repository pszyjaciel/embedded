\doxysubsection{Detailed Description}
Watchdog driver for the X\+MC microcontroller family. 

The watchdog unit (W\+DT) improves the system integrity, by triggering the system reset request to bring the system back from the unresponsive state to normal operation.

This L\+LD provides the Configuration structure X\+M\+C\+\_\+\+W\+D\+T\+\_\+\+C\+O\+N\+F\+I\+G\+\_\+t and initialization function \mbox{\hyperlink{group___w_d_t_ga44e875ad3a51f991024fc826dd309d62}{X\+M\+C\+\_\+\+W\+D\+T\+\_\+\+Init()}}.~\newline
It can be used to\+:
\begin{DoxyEnumerate}
\item Start or Stop the watchdog timer. (\mbox{\hyperlink{group___w_d_t_gae26358d03c9932008dcb663aefa47860}{X\+M\+C\+\_\+\+W\+D\+T\+\_\+\+Start()}} and \mbox{\hyperlink{group___w_d_t_ga8fd88b363d5f03cfe5a2907f683b2f82}{X\+M\+C\+\_\+\+W\+D\+T\+\_\+\+Stop()}})
\item Service the watchdog timer. (\mbox{\hyperlink{group___w_d_t_ga016fdcf06a9f9a9482f132ce8982fa24}{X\+M\+C\+\_\+\+W\+D\+T\+\_\+\+Service()}})
\item Configure the service window upper bound and lower bound timing values. (\mbox{\hyperlink{group___w_d_t_ga2033ba1f611f4639a8d6b167f4e5eb94}{X\+M\+C\+\_\+\+W\+D\+T\+\_\+\+Set\+Window\+Bounds()}})
\item Enable the generation of the pre-\/warning event for the first overflow of the timer. (\mbox{\hyperlink{group___w_d_t_gad70b8d499e7b43d7aeff503147c439e3}{X\+M\+C\+\_\+\+W\+D\+T\+\_\+\+Set\+Mode()}})
\item Clear the pre-\/warning alarm event. It is mandatory to clear the flag during pre-\/warning alarm I\+SR, to stop generating reset request for the second overflow of the timer. (\mbox{\hyperlink{group___w_d_t_gac85a3da7ce0fa894f2da3d8ce8c51a1b}{X\+M\+C\+\_\+\+W\+D\+T\+\_\+\+Clear\+Alarm()}})
\item Suspend the watchdog timer during Debug H\+A\+LT mode. (\mbox{\hyperlink{group___w_d_t_gafafe1be3b8e67ec8829d8b802e0a3251}{X\+M\+C\+\_\+\+W\+D\+T\+\_\+\+Set\+Debug\+Mode()}})
\item Configure service indication pulse width.(\mbox{\hyperlink{group___w_d_t_ga436569aaa25b7a1cdf9eeb8c1d272da2}{X\+M\+C\+\_\+\+W\+D\+T\+\_\+\+Set\+Service\+Pulse\+Width()}}) 
\end{DoxyEnumerate}
