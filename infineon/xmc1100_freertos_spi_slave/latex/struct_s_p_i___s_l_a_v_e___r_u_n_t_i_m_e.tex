Indicates the number of bytes to be used to represent one word of data. For values of word length less than or equal to 8, the size will indicate 1 byte. For rest of the values of word length, the size will be 2 bytes. This means that if word length is 10bits, each buffer access for transmission or reception will be done using 16bit access.
